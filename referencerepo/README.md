# Reference Repositories

This directory contains cloned reference repositories and research notes for the offline macOS/iOS vehicle-control Agent App.

- `repo_manifest.md`: selected upstream repositories and why they matter.
- `repo_urls.txt`: clone source list.
- `_scripts/clone_repos.sh`: repeatable clone/update script.
- `repos/`: local clones, ignored by project git workflows unless explicitly added later.
- `reports/`: per-repository research reports.
- `snapshots/`: extracted repository metadata used during research.
