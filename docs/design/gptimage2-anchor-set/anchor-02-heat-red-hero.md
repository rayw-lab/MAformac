## 4. 关键编码点 + 验收

- **制热红 ≠ 制冷蓝**：热浪图标（非雪花）+ 红橙渐变条 + 「制热」标签（SD20 核心，gptPRO 漏的）。
- **mode 双承载**：图标 + 下划线色都表制热（SD21#2）。
- **对话 inline 染色**：「28℃」暖红高亮（SD21#3）。
- **satisfied 态**：青辉光呼吸描边（D7，态色 ≠ 制热语义色，两层不冲突）。
- **5-gate**：hero 视觉最重、暖色统一、字清晰、数值主导。
