## 4. 关键编码点 + 验收（5-gate）

- **连续舞台**：三屏靠留白分块，**无 divider 黑线**（最易翻车）。
- **orb 主角**：96–112pt 无外框 + 星点粒子（不是小按钮）。
- **hero range bar**：大数值下方蓝渐变条 + 「制冷·自动」（SD20/SD21，比环形仪表更适 hero）。
- **次要族 fade**：靠下族卡淡显形成层级（SD21#7，但 10 族全可见）。
- **5-gate**：① 层级（hero 最重）② 对齐（8pt 栅格）③ 遮挡（无）④ 字体（清晰层级）⑤ 重量（数值主导）。
- **基线作用**：本张定调，A02-A11 上传它保持米白/连续舞台/玻璃/字体一致。
