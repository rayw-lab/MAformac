## 4. 关键编码点 + 验收

- **整屏边缘爆发**：这是磊哥反复强调的炸场核心（vivo 对标），不是 gptPRO 那种极轻 glow。
- **粒子飞溅**：边缘紫色粒子向内闪烁（SD17 氛围-4 Canvas 粒子）。
- **5s 高光帧**：本张是爆发瞬间定格（实装 5s 后 fade）。
- **克制**：紫光浓但不糊内容，高级不廉价（aesthetic gate）。
- **5-gate**：边缘主角、中间内容仍清晰、不遮挡。
