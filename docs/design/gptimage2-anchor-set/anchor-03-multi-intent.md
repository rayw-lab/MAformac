## 4. 关键编码点 + 验收

- **错峰非同时**：satisfied（空调）+ changing（座椅）两种态并存 = sequencer 依次点亮的定格（AD-4/SD19）。
- **态区分**：呼吸（satisfied 稳定）vs 脉冲（changing 流动）—— D7 两态视觉不同。
- **多 active 不强选单 hero**：两卡都强调，不放大成单一 hero 大卡（D2/SD19 场景1）。
- **5-gate**：双卡视觉重于背景族、节奏感、字清晰。
