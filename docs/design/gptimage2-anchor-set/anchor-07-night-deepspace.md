## 4. 关键编码点 + 验收

- **深空 ≠ 纯黑**：软黑 #121212（U11 halation 铁律，纯黑+高饱和 cyan = 散光糊屏）。
- **暗底辉光更炫**：orb / 边缘氛围 / satisfied 描边在暗底上对比强。
- **高对比文字**：近白蓝 #EAF0FF（暗底可读）。
- **halation 守门**：青色不铺满，大面积低透明。
- **5-gate**：层级清晰、文字可读（暗底投屏验收）、辉光不糊。
