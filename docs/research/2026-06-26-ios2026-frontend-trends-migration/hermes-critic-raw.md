# hermes-critic 一手回稿（调研计划批判）

## verdict
- 这个调研计划方向有价值，但最大风险是被"2026 新 UI / 新框架 / 新视觉系统"的叙事牵着走，把 MAformac 当前最需要的 demo 稳定性、离线端侧链路、5 分钟现场惊艳和可验证闭环，误替换成"追新技术清单"。
- 建议最终报告定位为"**面向 MAformac 的适配性筛选 + 低风险 spike 建议**"，不是"iOS 26/2026 前端升级路线图"。
- MAformac 当前阶段是方案演示助手/客户现场 demo，不是长期消费级 iOS App 设计系统重构；调研必须服务 L0-L3 验证门和可回滚落地。

## Top risks
1. **Liquid Glass / iOS 26 视觉系统 hype 风险**：新材质、透明、blur、玻璃感可能制造"高级感"错觉，但在座舱/投屏/低亮度场景可能牺牲可读性、遮挡安全、对比度和性能稳定。
2. **把"前端趋势"误当成"项目瓶颈"**：真实瓶颈可能在 ASR→LLM→指令→安全门→mock 端状态读回的可信闭环，而不是 UI 框架本身。
3. **官方叙事外推风险**：WWDC/HIG 是重要输入但仍是声称层；必须落到本 repo、部署 target、本机 build、截图/投屏可读性证据。
4. **与当前 OpenSpec 阶段错配**：若没有已对齐 change/spec，调研不应暗示马上实现；应输出 Adopt / Spike / Watch / Reject，而非直接推动改造。
5. **审美 gate 被"新系统质感"偷换**：新视觉不等于视觉合格；仍必须过层级、空间对齐、遮挡安全、字体可读、视觉重量 5 gate。

## 反方观点
- MAformac 要赢的是客户 5 分钟现场感知，不是技术博客里的 modern stack 分数。
- 一处稳定、可控、可解释业务价值的视觉增强，可能比大面积采用新组件更有效。
- 座舱语境下，过强玻璃拟态、过多动态、过薄字体会降低安全感和可信度。
- solo demo-tool 阶段不宜过度治理；除非组件/架构升级能直接降低后续状态页、多语言、演示场景制作成本，否则不该进主线。
- web researcher 的 source list 可能诱导结论；最终报告必须先写 MAformac 约束，再评估技术。

## 必须验证的问题
1. 当前 app 的最低/目标系统版本、Xcode/Swift 版本是否真的支持候选 API？不支持只能列 Watch/Future。
2. 候选视觉技术在 MAformac 核心屏幕上是否通过 5-gate，尤其遮挡安全和字体可读性？要截图/投屏证据，不能只看官方示例。
3. 新动画/材质是否影响冷启动、状态切换、ASR/LLM 反馈期间的响应稳定？demo 现场卡一次就是负收益。
4. 是否需要改状态管理/导航/组件体系？若要，收益是否大于回归风险？
5. 是否能用 isolated spike 验证：一个屏幕、一条状态链、一组截图、一个回滚点？不能 spike 的技术不适合当前阶段。
6. 是否影响离线端侧叙事？依赖云端、重型 SDK、远程素材、Python/非 iOS 运行时的方案应降级或排除。
7. 是否保持"mock 仅端状态"的语义？报告不得误写成 ASR/LLM/指令链路也 mock。
8. 技术是否提升客户理解"车控方案演示助手"的能力，而不是只提升截图审美？

## 最终报告应避免的误导
- 避免标题写成"iOS 26 全面升级建议"；建议写"面向 MAformac demo 的 2026 iOS UI 技术适配性筛选"。
- 避免把 Liquid Glass、SwiftUI 新 API、组件库趋势写成默认应采用；必须分 Adopt / Spike / Watch / Reject。
- 避免用"现代化、原生感、高级感"替代证据；每项建议要有 repo 切入点、验证方式、回滚方式。
- 避免把官方 demo/示例 app 效果外推到车控座舱场景；座舱优先读得清、状态稳、动效不抢语义。
- 避免预设当前 UI 架构落后；repo researcher 没给出真实瓶颈前，不要写"必须重构"。
- 避免把"2026-04-01 之后"当质量过滤器；新不等于成熟，旧但稳定的 SwiftUI/UIKit/HIG 基线仍可能是正解。
- 避免输出大而全路线图；当前合适产物是：现状约束 → 候选技术清单 → 适配性矩阵 → 1-2 个低风险 spike → 明确 reject 列表。

## 建议硬门
- 每个候选技术必须填 6 列：用户可见收益 / MAformac 场景匹配度 / 侵入范围 / 兼容与性能风险 / 验证证据 / 回滚成本。
- 没有验证证据的只能进 Watch，不得进 Adopt。
